Testing Multi-User Real-Time Collaboration

  1. Log in with different user accounts in each window below
  2. Navigate to the same patient list in both windows
  3. Edit the one_liner or HPI fields
  4. Observe real-time synchronization between the two users
  5. Try the following scenarios:
    • Type text in one window and see it appear in the other
    • Apply formatting (bold, italic, lists) and verify sync
    • Edit simultaneously in both windows
    • Use undo/redo and observe behavior
Note: For true multi-user testing, open this page in two different browsers or use incognito/private windows to maintain separate sessions. Each window should be logged in as a different user but accessing the same patient list.
